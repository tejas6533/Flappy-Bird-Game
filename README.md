# Flappy Bird Game

A modern web-based implementation of the classic Flappy Bird game with enhanced features and difficulty modes.

## Features

- **Circular Bird**: Your custom image (tejas.jpg) displayed as a circular bird
- **Difficulty Modes**: Choose from Easy, Medium, or Hard difficulty levels
- **High Score Tracking**: Persistent high score saved in your browser
- **Responsive Controls**: Play with spacebar or mouse clicks
- **Start Screen**: Clean game initialization with a start button
- **Game Over Screen**: Displays when you crash with restart option

## How to Play

1. Open `flappybird.html` in your web browser
2. Click "Start Game" to begin
3. Press spacebar or click to make the bird jump
4. Avoid the pipes and don't hit the ground or ceiling
5. Score points by passing through pipe gaps
6. Try different difficulty modes for varied challenges!

## Difficulty Levels

- **Easy**: Slower pipes, larger gaps, lighter gravity
- **Medium**: Standard challenge level
- **Hard**: Fast pipes, tight gaps, heavy gravity

## Controls

- **Spacebar**: Jump
- **Mouse Click**: Jump
- **Start Button**: Begin the game
- **Restart Button**: Reset the game anytime
- **Mode Buttons**: Switch difficulty (Easy/Medium/Hard)

## Technologies Used

- HTML5 Canvas for game rendering
- JavaScript for game logic
- CSS for styling
- Local Storage for high score persistence

## Files

- `flappybird.html`: Main game page
- `style.css`: Game styling
- `script.js`: Game logic and mechanics
- `tejas.jpg`: Bird image
- `README.md`: This file

## Running the Game

Simply open `flappybird.html` in any modern web browser. No server required - it's a client-side game!

## Browser Compatibility

Works in all modern browsers that support HTML5 Canvas:
- Chrome
- Firefox
- Safari
- Edge

## Customization

You can easily modify:
- Bird image: Replace `tejas.jpg` with your own image
- Difficulty settings: Edit the `difficulties` object in `script.js`
- Colors and styling: Modify `style.css`
- Game mechanics: Adjust values in `script.js`
